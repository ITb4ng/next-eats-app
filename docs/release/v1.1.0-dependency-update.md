# v1.1.0 의존성 업데이트 기록

작성일: 2026-05-25

기준 브랜치: `release/v1.1.0`

## 업데이트 기준

Next.js는 16 최신 버전으로 올리지 않고, Pages Router 안정성을 우선해 15 계열 최신 패치 버전으로 업데이트했다.

## 주요 변경

- `next`: `15.3.8` -> `15.5.18`
- `react`, `react-dom`: RC 버전 -> `19.2.6`
- `@prisma/client`, `@prisma/adapter-pg`, `prisma`: `7.6.0` -> `7.8.0`
- `axios`: `1.9.0` -> `1.16.1`
- `next-auth`: `4.24.11` -> `4.24.14`
- `pg`: `8.16.0` -> `8.21.0`
- `react-hook-form`, `react-icons`, `react-select`, `react-toastify`, `zustand` 패치/마이너 업데이트
- `@types/react`, `@types/react-dom`을 React 19 타입으로 정렬
- 사용하지 않는 `install`, `recoil` 제거
- `dev:all` 스크립트에 필요한 `concurrently` 추가

## 보안 대응

`yarn audit --groups dependencies` 기준 운영 의존성 취약점이 0건이 되도록 조정했다.

직접 의존성 업데이트만으로 해결되지 않는 하위 패키지는 `yarn resolutions`로 패치 버전을 고정했다.

- `brace-expansion`
- `minimatch`
- `postcss`
- `preact`
- `uuid`
- `yaml`

## 코드 보정

React 19 타입 기준에 맞춰 `useIntersectionObserver`가 `RefObject<Element | null>`을 받을 수 있도록 수정했다.

## 검증 결과

```bash
npm run lint
npm run build
yarn audit --groups dependencies
```

결과:

- lint 성공
- build 성공
- 운영 의존성 audit 0건

## 남은 경고

- `next lint`는 Next 16에서 제거될 예정이므로 후속 작업에서 ESLint CLI로 전환한다.
- `react-query v3`는 React 19 peer dependency 경고가 남는다. v1.1.0에서는 동작 안정성을 우선해 유지하고, 후속 버전에서 `@tanstack/react-query` 전환을 검토한다.
- `resolutions`로 보안 패치 버전을 고정한 일부 하위 패키지에서 호환성 경고가 표시된다. lint/build와 audit은 통과했다.
