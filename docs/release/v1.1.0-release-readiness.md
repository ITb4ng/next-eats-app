# v1.1.0 릴리즈 준비 문서

작성일: 2026-05-25

기준 브랜치: `dev`

## 릴리즈 목표

v1.1.0은 지도 장애 대응 이후 사용자가 지도에만 의존하지 않고도 Store 목록에서 탐색을 이어갈 수 있도록 만든 안정화 릴리즈다. 핵심은 `/stores` 탐색 경험을 강화하고, 지역 필터와 검색어, 지원금 사용 가능 여부를 함께 사용할 수 있게 만드는 것이다.

## 최소 완성 기준

- 지도 로드 실패 시에도 `/stores`로 이동해 탐색을 이어갈 수 있다.
- 지도 실패와 Store API/DB 실패 상태가 분리되어 있다.
- Store 데이터에 `acceptsPaySupport`를 저장하고 생성/수정/조회 흐름에서 유지할 수 있다.
- 맛집 등록/수정 화면에서 지원금 사용 가능 여부를 설정할 수 있다.
- Store 목록/상세/StoreBox에서 지원금 사용 가능 여부를 확인할 수 있다.
- 지원금 사용 가능 매장만 보기 필터가 지역 필터, 검색어 필터와 함께 동작한다.
- 지역 검색 필터가 서울/춘천 중심에서 전국 시/도 단위로 확장되어 있다.
- Store 목록, StoreBox, 등록, 수정, 찜, 댓글 UI/UX가 v1.1.0 기준으로 정리되어 있다.
- 릴리즈 전 확인 항목과 후속 분리 항목이 문서화되어 있다.
